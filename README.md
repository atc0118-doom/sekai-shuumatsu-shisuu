# 終末観測盤｜リアルタイム半自動投稿版 v2

## 変更点

ウクライナ・ロシアなど、報道量が多い地域が常に100へ張り付く問題を修正しました。

新しい採点方式：

- 地域ごとの平常リスクを設定
- ニュース件数より「前回比の変化」を重視
- 核・ミサイル・封鎖・空爆など重大語を重く評価
- 停戦・支援・協議などは少し減点
- 重大語がない通常報道では高得点に張り付きにくいソフトキャップを追加

## 使い方

1. ZIPを解凍
2. GitHubの既存リポジトリへ `index.html` を上書きアップロード
3. Commit changes
4. Vercelが自動デプロイ
5. サイトを開いて確認

## 注意

この指標は公開ニュース情報をもとにした独自観測値です。予測や安全保障判断を保証するものではありません。
