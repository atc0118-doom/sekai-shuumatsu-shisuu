# 終末観測盤 / 世界終末指数

日本語おどろおどろしい版 + Google Analytics 対応です。

## Google Analytics を使う

`index.html` の上部にある次の行を探してください。

```js
window.WDI_GA_ID = 'G-XXXXXXXXXX';
```

`G-XXXXXXXXXX` を自分の GA4 測定IDに置き換えると、アクセス数・国・流入元などを Google Analytics で確認できます。
