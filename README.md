# 終末観測盤｜リアルタイム半自動投稿版 v3

## 変更点

関連ニュースを英語見出しのまま出すだけでなく、サイト側で日本語の短い要約を自動生成するようにしました。

追加内容：

- 関連ニュース欄に日本語要約を表示
- 投稿文生成にも日本語ニュース要約を混合
- 元記事リンクはそのまま残す
- スコアはv2の改善版を維持
  - ニュース件数だけで100に張り付きにくい
  - 前回比の変化量を重視
  - 核・ミサイル・封鎖・空爆など重大語を加点
  - 停戦・協議・支援は少し減点

## 使い方

1. ZIPを解凍
2. GitHubの既存リポジトリへ `index.html` を上書きアップロード
3. Commit changes
4. Vercelが自動デプロイ
5. サイトを開いて「関連ニュース」に日本語要約が出るか確認

## 注意

この日本語要約は、ブラウザ内で見出しキーワードから作る簡易要約です。外部AI APIキーは不要です。全文翻訳ではありません。

より自然なAI要約にしたい場合は、次段階でOpenAI APIなどをVercel Serverless Function経由で接続してください。

この指標は公開ニュース情報をもとにした独自観測値です。予測や安全保障判断を保証するものではありません。
